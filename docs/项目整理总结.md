## 四、熵权法权重分析

### 4.1 方法说明

熵权法是一种基于信息熵的客观赋权方法，通过数据的离散程度来确定各指标的权重。

### 4.2 计算步骤

1. 数据标准化（极差标准化）
2. 计算比重 P\_ij
3. 计算熵值 e\_j
4. 计算差异系数 d\_j = 1 - e\_j
5. 计算权重 w\_j = d\_j / sum(d\_j)
