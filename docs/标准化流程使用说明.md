## 概述

本标准化流程实现了完整的经济-环境耦合协调度分析，包括：

| 步骤      | 功能         | 核心模块                             |
| ------- | ---------- | -------------------------------- |
| **步骤1** | 数据预处理      | `DataPreprocessor`               |
| **步骤2** | 熵权法权重计算    | `EntropyWeightCalculator`        |
| **步骤3** | TOPSIS综合得分 | `TOPSISCaculator`                |
| **步骤4** | 耦合协调度      | `CouplingCoordinationCalculator` |
| **步骤5** | 时间演化可视化    | `Visualizer`                     |

***

## 数据格式要求

### 标准数据字段

数据文件必须包含以下字段：

| 字段名       | 说明          | 单位      | 类型 | 指标方向 |
| --------- | ----------- | ------- | -- | ---- |
| `年份`      | 数据年份        | -       | 整数 | -    |
| `省份`      | 省份名称        | -       | 文本 | -    |
| `人均GDP`   | 人均地区生产总值    | 元       | 数值 | 正向   |
| `第三产业占比`  | 第三产业增加值占比   | %       | 数值 | 正向   |
| `R&D占比`   | 研究与试验发展经费占比 | %       | 数值 | 正向   |
| `单位GDP能耗` | 单位GDP能源消费量  | 吨标准煤/万元 | 数值 | 负向   |
| `单位GDP水耗` | 单位GDP水消费量   | 立方米/万元  | 数值 | 负向   |
| `污水处理率`   | 污水处理率       | %       | 数值 | 正向   |
| `垃圾处理率`   | 垃圾处理率       | %       | 数值 | 正向   |
| `森林覆盖率`   | 森林覆盖率       | %       | 数值 | 正向   |
| `优良天数比例`  | 空气质量优良天数比例  | %       | 数值 | 正向   |
| `环保投资占比`  | 环保投资占GDP比例  | %       | 数值 | 正向   |

### 支持的文件格式

- **CSV格式**: `.csv`，UTF-8-BOM编码
- **Excel格式**: `.xlsx`

## 模块详解

### 1. 数据预处理模块 (DataPreprocessor)

#### 1.1 加载数据

```python
from standard_analysis import DataPreprocessor

df = DataPreprocessor.load_data('data.xlsx')
```

#### 1.2 缺失值处理

```python
df_clean = DataPreprocessor.clean_missing_values(df)
```

**处理策略**:

- 时间序列数据：线性插值
- 非时间序列数据：省份均值
- 兜底策略：全局均值

#### 1.3 异常值检测

```python
df_clean, outliers_info = DataPreprocessor.detect_outliers(
    df, method='both'
)
```

**检测方法**:

- `method='iqr'`: 箱线图法（IQR）
- `method='zscore'`: 3σ原则
- `method='both'`: 双重检测（默认）

#### 1.4 极差标准化

```python
df_normalized, norm_info = DataPreprocessor.min_max_normalization(
    df,
    positive_indicators=['人均GDP', '第三产业占比', ...],
    negative_indicators=['单位GDP能耗', '单位GDP水耗']
)
```

**标准化公式**:

- 正向指标: X' = (X - X\_min) / (X\_max - X\_min)
- 负向指标: X' = (X\_max - X) / (X\_max - X\_min)

***

### 2. 熵权法权重计算 (EntropyWeightCalculator)

#### 2.1 计算权重

```python
from standard_analysis import EntropyWeightCalculator

weights, intermediate = EntropyWeightCalculator.calculate_entropy_weight(
    normalized_matrix
)
```

**计算流程**:

1. 计算比重矩阵: P\_ij = X\_ij / ΣX\_ij
2. 计算信息熵: E\_j = -kΣ(P\_ij ln P\_ij)
3. 计算差异系数: g\_j = 1 - E\_j
4. 计算权重: W\_j = g\_j / Σg\_j

#### 2.2 获取子系统权重

```python
subsystem_weights = EntropyWeightCalculator.get_subsystem_weights(
    weights,
    all_indicators,
    economic_indicators,
    environmental_indicators
)
```

***

### 3. TOPSIS综合得分 (TOPSISCaculator)

#### 3.1 计算综合得分

```python
from standard_analysis import TOPSISCaculator

S, topsis_intermediate = TOPSISCaculator.calculate_topsis(
    normalized_matrix,
    weights,
    positive_indicators,
    all_indicators
)
```

**计算流程**:

1. 构建加权标准化矩阵: V\_ij = W\_j × X\_ij
2. 确定正/负理想解
3. 计算距离
4. 计算综合得分: S\_i = D\_i- / (D\_i+ + D\_i-)

#### 3.2 子系统得分

```python
subsystem_scores = TOPSISCaculator.calculate_subsystem_scores(
    normalized_matrix,
    subsystem_weights,
    positive_indicators,
    economic_indicators,
    environmental_indicators,
    all_indicators
)
```

***

### 4. 耦合协调度计算 (CouplingCoordinationCalculator)

#### 4.1 计算耦合协调度

```python
from standard_analysis import CouplingCoordinationCalculator

coupling_results = CouplingCoordinationCalculator.calculate_coupling_coordination(
    U1, U2,
    alpha=0.5,
    beta=0.5
)
```

**计算流程**:

1. 耦合度: C = 2√(U1×U2)/(U1+U2)
2. 综合发展指数: T = 0.5×U1 + 0.5×U2
3. 耦合协调度: D = √(C×T)

**协调等级**:

- D ≥ 0.8: 优质协调
- 0.6 ≤ D < 0.8: 中度协调
- 0.4 ≤ D < 0.6: 勉强协调
- 0.2 ≤ D < 0.4: 中度失调
- D < 0.2: 极度失调

import pandas as pd

# 创建省份列表
provinces = [
    '北京', '天津', '河北', '山西', '内蒙古',
    '辽宁', '吉林', '黑龙江', '上海', '江苏',

## 常见问题

### Q1: 如何只运行部分步骤？

```python
from standard_analysis import DataPreprocessor, EntropyWeightCalculator

# 只做数据预处理
df = DataPreprocessor.load_data('data.xlsx')
df_clean = DataPreprocessor.clean_missing_values(df)
df_normalized, _ = DataPreprocessor.min_max_normalization(df_clean, ...)

# 只计算权重
weights, _ = EntropyWeightCalculator.calculate_entropy_weight(
    df_normalized.values
)
```

### Q2: 如何自定义指标？

修改 `standard_analysis.py` 顶部的常量定义：

```python
# 自定义正向指标
POSITIVE_INDICATORS = ['指标1', '指标2', ...]

# 自定义负向指标
NEGATIVE_INDICATORS = ['指标3', '指标4', ...]

# 自定义子系统
ECONOMIC_SUBSYSTEM = ['指标1', '指标2']
ENVIRONMENTAL_SUBSYSTEM = ['指标3', '指标4']
```

### Q3: 如何修改协调等级标准？

修改 `COORDINATION_LEVELS` 常量：

```python
COORDINATION_LEVELS = [
    (0.9, 1.0, '优质协调'),
    (0.75, 0.9, '良好协调'),
    (0.6, 0.75, '中度协调'),
    (0.4, 0.6, '勉强协调'),
    (0.2, 0.4, '轻度失调'),
    (0.0, 0.2, '严重失调')
]
```

### Q4: 如何添加新的可视化图表？

继承 `Visualizer` 类添加新方法：

```python
from standard_analysis import Visualizer

class CustomVisualizer(Visualizer):
    @staticmethod
    def plot_custom_chart(df_result, save_path):
        # 自定义图表实现
        pass
```

### Q5: 内存不足怎么办？

对于大数据集，可以：

1. 分批处理数据
2. 使用更高效的数据类型
3. 只处理需要的列
4. 使用Dask或Vaex等库
