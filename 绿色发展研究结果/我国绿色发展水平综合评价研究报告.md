### 1.1 研究背景

### 1.2 研究方法

#### 1.2.1 数据标准化

- **正向指标**: $X'_{ij} = \frac{X_{ij} - \min(X_j)}{\max(X_j) - \min(X_j)}$
- **负向指标**: $X'_{ij} = \frac{\max(X_j) - X_{ij}}{\max(X_j) - \min(X_j)}$

#### 1.2.2 熵权法

1. 比重矩阵: $P_{ij} = \frac{X_{ij}}{\sum_{i=1}^{m} X_{ij}}$
2. 信息熵: $E_j = -k \sum_{i=1}^{m} P_{ij} \ln(P_{ij})$
3. 差异系数: $G_j = 1 - E_j$
4. 权重: $W_j = \frac{G_j}{\sum_{j=1}^{n} G_j}$

#### 1.2.3 TOPSIS方法

1. 加权标准化矩阵: $V_{ij} = W_j \times X_{ij}$
2. 正/负理想解: 确定各指标最优值和最差值
3. 欧氏距离: $D_i^+ = \sqrt{\sum_{j=1}^{n} (V_{ij} - V_j^+)^2}$
4. 综合得分: $S_i = \frac{D_i^-}{D_i^+ + D_i^-}$

#### 1.2.4 耦合协调度

1. 耦合度: $C = 2 \times \sqrt{\frac{u_1 \times u_2}{u_1 + u_2}}$
2. 综合发展指数: $T = 0.5u_1 + 0.5u_2$
3. 耦合协调度: $D = \sqrt{C \times T}$
